Sistema Help desk

Olá! Este é um projeto desenvolvido para a disciplina de Usabilidade, Desenvolvimento Web, Mobile e Jogos.
Trata-se de um sistema de Help Desk, uma plataforma centralizada para gerenciar e solucionar solicitações de suporte dos usuários, otimizando o atendimento.
O projeto foi construído com Angular 12 no front-end e Spring Boot 2 no back-end, utilizando conceitos como aplicações de página única (SPA) e autenticação com Tokens JWT.
